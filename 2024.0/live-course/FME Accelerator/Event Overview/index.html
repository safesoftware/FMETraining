Overview

Join our free and interactive workshop to learn how to get the most out of your data with FME. This workshop is for users beginning their FME journeys and for users refreshing their FME knowledge. The FME Accelerator will show you the basics of automating your data integration workflows and connecting to 500+ formats. Spend more time using data and less time fighting it.

We strongly recommend signing up and attending the event live. Our partners offer the FME Accelerator in other time zones and languages. 

If you cannot attend live, request a virtual machine and watch the course recording instead. Please email train@safe.com with any questions.

Event Technical Requirements

We host this course using a live training platform called Strigo. You will access Strigo through the FME Academy and do not need to create any additional accounts.

Please confirm your system is compatible with Strigo by running these precourse tests. 

You may encounter technical difficulties if you try to access Strigo on a VPN or behind a corporate firewall, even if you pass the precourse tests. We recommend turning these off when attending or contacting your IT department and providing them with the information below to whitelist Strigo.

For a smoother and more productive experience, we recommend you have a second monitor or a second device like a tablet so you can work on exercises from one screen and view the instructions on the other. But this is not required.

Strigo

Strigo is a fully managed solution that runs in the browser, so no downloads or installations are needed. We provide FME for you for the duration of the event. There are, however, some prerequisites for ensuring a smooth experience.

Base requirements

  • A stable internet connection.
  • A supported browser (Chrome v51+, Mozilla Firefox).
  • A screen resolution of at least 1280x720.

Note: You can check which browser you're running here.

Audio/ Video requirements

We highly recommend going through this pre-course system test.

Note: You must run the test from the same computer and network you plan to use during the event and use the same camera and microphone.

At the time of the event, we recommend following these guidelines:

  • Ensure you don't have other applications running using your media devices (microphone, camera, or speakers).
  • Disable any ad blockers or script blockers.
  • The required minimum bandwidth is 300kbps for video and 50kbps for audio.

Network requirements

We recommend going through the list below with your organization's IT admin. Please note that Strigo will likely not work if you are using a VPN or behind a firewall. We recommend disabling those before accessing the event.

Make sure that:

  • Port 443 is open.
  • WebSocket traffic is allowed in the organization's firewall (ws and wss).

Note: this tool can check whether the port is opened and WebSocket traffic is allowed.

The following domains must be accessible to you:

  • *.strigo.io (You can check that it's available by going to https://app.strigo.io)
  • s3-eu-west-1.amazonaws.com
  • *.googleapis.com
  • *.gstatic.com
  • *.opentok.com
  • *.tokbox.com
  • *.firebase.com
  • *.firebaseio.com

Lab Bandwidth Requirements

The required bandwidth is expected to be around 5-10mbps (mbit).
